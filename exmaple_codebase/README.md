# Example codebase (for codebase2context)

This folder is a tiny, intentionally simple “fake repo” used to demonstrate what `codebase2context.py` generates.

## Regenerate the sample context

From the repository root:

```bash
cd exmaple_codebase
python codebase2context.py
```

This writes/overwrites `exmaple_codebase/CODEBASE_CONTEXT.md`.
